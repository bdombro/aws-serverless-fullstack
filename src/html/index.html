SERVERLESS-FULLSTACK

Welcome to your AWS serverless application. This example application has several resources configured for you to explore.

Resources

GET /db-ping

 Checks if the database is ready

curl 0.0.0.0:8080/db-ping

POST /files/:id

 Uploads a file.

curl 0.0.0.0:8080/files/tmp.jpg --form file=@image.jpg

GET /files/:id

 Returns a file

curl 0.0.0.0:8080/files/__ping__

GET /files/:id/meta

 Returns a file's metadata

curl 0.0.0.0:8080/files/__ping__/meta -H 'accept: application/json'
